# websocket-experiments

Some experiments with web sockets running on Google Cloud Platform.

https://chat-gcp-6tl7g2wh6a-uk.a.run.app/

## Run locally

```
cd websocket-experiments/chat-gcp

npm install

npm start
```

## Deploy to Google Cloud Platform

```
cd websocket-experiments/chat-gcp

gcloud run deploy
```
