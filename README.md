# websocket-experiments

Some experiments with web sockets running on Google Cloud Platform.

https://gcp-chess-6tl7g2wh6a-uk.a.run.app/

## Run locally

```
cd websocket-experiments/gcp-chess

npm install

npm start
```

## Deploy to Google Cloud Platform

```
cd websocket-experiments/gcp-chess

gcloud run deploy
```
